We want to be able to draw rectangles for a board given the number of columns and rows and the range of x and y values that the board should cover. In this step you're going to make the following block that does just that. Notice below that both examples draw in the same range, but there are different numbers of rows and columns in each.






The variables that we used to make this were rather complicated.

Input to the function:
  • width - number of columns desired
  • height - number of rows desired
  • x1 - lower left x position where the board should start
  • y1 - lower left y position where the board should start
  • x2 - upper right x position where the board should start
  • y2 - upper right y position where the board should start
Script variables:
  • pixel height - the total number of pixels between x1 and x2
  • pixel width - the total number of pixels between y1 and y2
  • square height - the height (in pixels) of the game board squares
  • square width - the width (in pixels) of the game board squares
