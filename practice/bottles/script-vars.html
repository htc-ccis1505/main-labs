Variables

A variable is a placeholder in your code. It is a place to store a value so that you can come back to it or use it multiple times. As you might expect from the name, it is variable meaning that its value can change.

There are two different types of variables in Snap: script variables and global variables. For this exercise we will focus on script variables.

Script Variables

In Lab 3: Script Variables, we learned how to make and use a script variable.

A script variable is only usable in the script where it is made. It cannot be used in another script, even within the same sprite.


To use a script variable inside a custom block, you must pass it in as input to that block.


Set and Change

To set a variable, you use the set block. This gives the variable a specific value. The change block alters (adds or subtracts from) the existing value by the amount entered. The show and hide blocks allow you to see the variable's value and can be helpful for debugging your script.
