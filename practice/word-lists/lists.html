In Lab 5: Lists and Higher Order Functions (HOFs), we learned how to make and use lists.

List Blocks

Recall that a list is a way to combine several values into one grouped value. It is an object that allows you to store a series of values inside one variable.

snap gives us many different blocks to use for working with lists. In this practice we will focus on using the list block to create a list, the item [#] of [list] block, the length of [list] block, the add [] to [list] block, and the delete [#] from [list] block.

Get a List Item

The item [] of [list] reporter block is used to get an item from a list. You use the drop-down to select either item 1, item last, or item any - a random item. You can also type in a number or use a variable to get an item from a different position in the list.
