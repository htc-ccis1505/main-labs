Removing List Items

The delete [#] of [list] block is used to remove an item from a list. You use the drop-down to select either item 1, last, or all. You can also type in a number or use a variable to remove an item from a different position in the list.

There's also a more subtle problem that the loop runs until i is equal to the original length of the list. You can see this by noticing that the script glows longer than the last say.

To fix this, we need to use a different loop where we control the counter and the number of times the loop runs is conditional - a repeat until with a separate script variable is perfect for this.


When removing items from a list, do not use a for loop and make sure the counter is adjusted as items are removed from the list. Use good test data to ensure no values are missed.
