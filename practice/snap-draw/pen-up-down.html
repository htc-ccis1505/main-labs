At this point, your script should look like what you see below.


When you click the Green Flag, the sprite should follow your mouse around the stage area. It should continue to follow the mouse until you click the Red Stop icon.

Pen Up and Down

The next step is to have the pen go up and down when the arrow keys are pressed. For this, we need more scripts, each starting with their own hat-shaped block. Locate the When [space] key pressed block on the Control palette. Notice that you can change which key it responds to by using the drop down next to [space].

Add one When [space] key pressed block to respond to the up arrow key. Add the Pen up block to this script. That block is found on the Pen palette.

Add another When [space] key pressed block to respond to the down arrow key. Add the Pen down block to this script. That block is also found on the Pen palette.

Try to get this part of the script working without peeking at the answer on the next page. When you are done, you should be able to click the green flag to have the sprite follow the mouse around the stage area. When you push the down arrow and then move the mouse, it should draw. When you push the up arrow it should stop drawing.
