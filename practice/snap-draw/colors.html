At this point, your script should look like what you see below.


When you click the Green Flag, the sprite should follow your mouse around the stage area. When you push the down arrow and then move the mouse, it should draw. When you push the up arrow it should stop drawing.

Add Colors

The next step is to have change the pen color when different keys are pressed. Find the set pen color to [] block on the Pen palette. We will use this to change the color when different keys are pressed.

There is a color picker on the set pen color to [] block which you can access by clicking on the colored square.

Make scripts using the When [space] key pressed and set pen color to [] block to do the following:
- when the R key is pressed, set it to a red color
- when the G key is pressed, set it to a green color
- when the B key is pressed, set it to a blue color
- when the Y key is pressed, set it to a yellow color
- when the P key is pressed, set it to a purple color
- when the O key is pressed, set it to an orange color

Try it out. When you press the letter keys, you should be able to draw in different colors! Remember to put your pen down first.
