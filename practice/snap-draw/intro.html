Program Flow

To understand how to put the pieces together, it is important to remember that the When hat-shaped blocks from the Control palette, control when a script will start running. When that script is started, then it will run each block in order, sequentially, from top to bottom.

Loops can be used to repeat steps in that sequential flow. Think about the instructions for shampooing your hair:
- Wet hair
- Apply shampoo
- Massage into hair to lather
- Rinse and repeat

This "repeat" means to go back up to the top of the steps and do it again from the beginning. Let's program this in snap.
