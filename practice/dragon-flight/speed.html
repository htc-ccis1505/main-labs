Speed Control

If you have a value of 10 in your move block, your dragon might be moving fairly quickly around the screen. We can change the speed of flight by altering the value in the move block. We'll use a variable for this and then alter the speed with the up and down arrow keys.

Script Variables were introduced in Lab 3: Script Variables and global variables were introduced in Lab 4: Global Variables.

Create a new global variable named "speed". Because we will use this across multiple scripts, we can't use a script variable here. Add a set [var] to [] block to set speed to 5. Then update the move block at the bottom to use the speed variable.


Next we can use the when [] key pressed and change [var] by [] blocks to increase and decrease the speed.


If you try this out however, you might notice that you can make the dragon fly backwards, and you can also get the dragon moving very fast. This isn't good. We need a to limit the max and min speed.
