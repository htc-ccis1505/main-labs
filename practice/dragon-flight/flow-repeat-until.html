Conditional Program Flow

One way to conditionally alter flow is by using the if or if-else blocks. We can also have a loop repeat conditionally. The repeat until block allows us to loop until a condition is true. We repeat the loop while the condition is false, and stop when it becomes true.

Example

Often when we ask for user input we want to do some verification that what they enter is what we expect, and if it isn't we'll ask again. Let's ask the answer to a simple question: What is 2 + 2? Then we will make sure that the person enters a number by repeating the ask until a number is entered.
