Add Costumes

We're going to fly a dragon around the screen, not a turtle, so we need to import the dragon costumes from the menu. Click the file icon and select Costumes...


Select the dragon1-a.png from the list. Then repeat this and select dragon1-b.png from the list.

Next go to the costumes tab. It is behind the scripts tab in the center of the page.


You should see the turtle sprite and below it two dragon images. If you do not see the dragon costumes, repeat the steps above. If you click on one of the images, the sprite's appearance will change to that image.

Rotation Style

We have a cool dragon sprite now, but we need to change his movement style to prevent him from upside down flight.

Next to the sprite image, above the tabs in the center of the screen, you'll see 3 small round buttons.
- The top button allows 360 degree rotation of the sprite.
- The middle button only allows the sprite to face left and right. This is the option we want.
- The bottom button prevents the sprite from rotating at all.


Code for Costume Change

There is a switch costume [] block on the Looks palette. Use this block to change the sprite to the dragon1-a.png costume when the Green flag is clicked.

Then when the space key is pressed:
- change to the dragon1-b.png costume
- wait for 2 seconds (the wait block is on the Control palette)
- switch back to the dragon1-a.png costume.
