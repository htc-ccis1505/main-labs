Variables

Recall that a variable allows you to name and store and change a value. Variables have what is called a scope, which indicates where the variable is visible and can be used. The scope of a script variable is the script that it is defined in. The scope of an input parameter is the block (or function) that it is part of.

Global Variables

In Lab 4: Script Variables, we learned how to make and use a global variable.

When you create a global variable, you can choose to make it available to your entire program (all sprites) or limit it to just the current sprite. This is defining the scope of the global variable. For this class, keep the default of all sprites. Global variables defined in this way can be used in all scripts, including within custom blocks (functions) without passing in the value as input.
