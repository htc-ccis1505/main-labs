Save a Project

To save your snap project to the snap Cloud, click on the file icon in the window toolbar, and then click on the "Save" option.

In the Save Project window, make sure the cloud button is highlighted in blue on the left hand side of the window. Name your project in the space for you type in a title. Afterwards, click on the save button.

Export Project

To save your snap project to your computer, click the file icon in the toolbar and select the option to "Export project...". This is how you will turn in your snap assignments to D2L.

Note that this does NOT work correctly in the Internet Explorer web browser. You need to use Google Chrome or Firefox to work with snap and to export your projects.

Share a Project

Once your project is saved in the cloud, you can share it by clicking the Share button at the bottom of the Save Project window. Shared projects are listed in bold in the project list. Once a project is shared, you can unshare it by clicking the Unshare button.

Open a Shared Project

To open a shared project, you need to hold down the Shift key while you click the cloud icon in the toolbar. This will add red items to the menu, including the option to "open shared project from cloud...".

To view a shared project, you need to have the Snap user name of the person sharing the project and the exact project name.
