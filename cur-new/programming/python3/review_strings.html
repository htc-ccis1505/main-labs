Review Strings

Recall that a string is just a sequence of characters, like a word or a sentence. To create a string, we set a variable to text enclosed in quotes (single or double) as shown below: 


>>> phrase = "Programming is awesome!"
>>> cat = 'Bart'

Remember when you see the >>> in the examples, that indicates the example is entered into the python interactive shell. Do NOT type the >>> into your python file.

String Length

We can get the length of a string by using the len() function. This function returns (or reports in snap speak) back a value. To do something with that value, you can use it to set a variable. 


>>> len(phrase)
23

The len() function returns (or reports in snap speak) back a value. To do something with that value, use it to set a variable. Then you can use that variable in other statments such as a print or a loop. 


>>> phrase_length = len(phrase)
>>> print( "The length of the phrase is " + str(phrase_length) )
The length of the phrase is 23

Notice that in the code above the + operator was used to join strings together.

Characters and Substrings

We can access certain letters of a string by using square braces []. 


>>> first_letter = phrase[0]
>>> print( "The first letter of the phrase is " + first_letter )
The first letter of the phrase is P

Common Mistake:
Remember that in python the first letter of a string is at position 0, not position 1.
Note: Having the first element at the 0th position is called "0 indexing" and is very common in many programming languages.

We can get at parts of a string (or a substring) by using square braces with a colon. Substring [start:end] returns the letters in the word from start up to, but not including the character at end. 


>>> part = phrase[3:7]
>>> print( part )
gram
