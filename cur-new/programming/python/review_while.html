While Loop

The while loop is commonly used in many programming languages. It is similar to the 'repeat until' block in snap, however in Python the loop ends when the condition becomes False. This is opposite of 'repeat until', which waits for its condition to become True. 


i=0
while i < 5:
  print(i)
  i += 1
print( "Loop is finished!" )

Notice that the line with the while ends with a : and that the following lines are indented. The indentation indicates what code is repeated by the loop. The next line that is not indented will happen when the loop is over.

Spacing and formatting in Python is extremely important and mixing tabs and spaces can be confusing. Do NOT use the tab key unless your editor will convert them to spaces for you. Many programming editors will do this for you automatically.

Loops and Strings

We learned that we can get the length of a string using the len(my_string) function and that we can get a single character from a string using square braces and the character position my_string[position]. We can use these together to make a while loop that will print each letter of a word one at a time.
