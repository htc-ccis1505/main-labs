Return Values

In snap speak) we made reporter blocks and predicate blocks that could report back a value. In python, functions that we write are also able to report back values, but we say that the function returns a value instead of reports a value. This corresponds to the word return which is used in python (and many other languages). 


>>> def add(x, y):
...    return x + y

>>> print add(2,3)
5

Find Exercise 2 in the functions.py file and add the code to the exponent(num, power) function that has been defined for you. Notice that it takes two arguments (a number and an exponent value) and returns the computed result. Below are a few example inputs. 


>>> exponent(10, 0)
1
>>> exponent(5, 3)
125
>>> exponent(2, 10)
1024

For exponent, any value with 0 as the exponent should give you the result of 1. For any other exponent, that tells you how many times to multiply the number by itself. So exponent(5, 3) = 5 * 5 * 5 = 125.
