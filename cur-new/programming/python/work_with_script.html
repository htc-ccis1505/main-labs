Running the Lab Script

Get the starter file

Now it's time to seriously write some code. Start by right clicking on this link to get the welcome.py starter file and select save as to save it to a directory on your computer.

Use the cd command to go to the directory where you saved the file. Then use the ls command to verify that you are in the correct directory with the welcome file. 


C:\Users\Staff> cd C:\myDirectory
C:\myDirectory> ls
welcome.py

Edit the File

We will be editing the python file in a text editor. There are many text editors available, any will do. In this lab we'll be using Brackets which should be installed the lab computers, and it is free to install on your home computers as well. Another good, free option is Atom

First, open up your editor and then open welcome.py. Make sure to read the comments at the top of the file. Then set the variable called your_name just below the comments. For example, your_name = "Alonzo" (make sure your name is inside quotes as this is how Python designates text).

Head back to the shell and you can try running this file by entering python welcome.py as shown below: 


C:\myDirectory> python welcome.py

At this point, all it will do is welcome you by name. (Unless you have forgotten to update the your_name variable...) Later you will complete the lab by adding code below each comment.
