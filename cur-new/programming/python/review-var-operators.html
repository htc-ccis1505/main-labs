Review Variables

Remember in Python, we don't explicitly declare (or make) a variable in code. They are created when they are assigned to a certain value: 


x = 18
y = 24
z = x + y

Remember that in Python you cannot have spaces in your variable or function names. Use underscores to separate words my_variable or camelCase myVariable.

To change the value of a variable, you assign it a new value with the equal operator. Creating a variable and changing the value of a variable look the same in python. 


x = 1
x = x + 1

Remember that in Python x = x + 1 is the same as x += 1.
