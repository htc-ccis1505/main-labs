There are two essential kinds of loops in most programming languages: the For-Loop and the While-Loop. We already know how to write a while loop in python, next we'll look at how to write a for-loop. Here is a simple for loop in snap side-by-side with the same loop in python. 


for i in range( 1, 10 ):
   print( i )

This for-loop will print the value of i, 10 times. The value of i starts at 1, and each time through the loop, the value of i goes up by 1.

Range Function

The for loop in Python works similarly to our snap for loop with some key differences. The python for loop uses a range function create an iterable. An iterable is a just list of items. The range(x, y) function receives two numbers and returns a list of all numbers beginning with x and ending with y-1. In this example, the list of items starts at 1 an goes up to - but does not inlcude - 10. Starting with the first item, i takes each consecutive item of the list as its value.

You can also write the range function with one argument, which causes it to start at 0. The following example will start at 0 and go up to (but not include) 10. 


for i in range( 10 ):
   print( i )

Try it! These examples use the for loop and range to count up 9. Modify the example to count down from 10.
