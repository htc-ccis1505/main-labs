So far you've seen how to run python from the command line (also known as a shell) via the interpreter; however, we can also write our code in a separate text file, which we can save and then run from the command line. This is how most people create python programs. However, before we can start writing more Python, it is helpful need to learn a few shell commands.

Some of the commands introduced here are different between a Windows computer and Mac OS X or Unix/Linux computer.

How files are organized on a computer:

As you probably already know, your python files, and any file for that matter, can be found within a particular folder (also know as a directory) on your computer in a structure similar to the one shown below:


To navigate to a particular python file we may need to make use of the following shell commands. (Whenever we say "enter a command" this means type the command and then press enter to execute it).

  1. The command dir (or on Mac ls) allows you to view a list of all of the files and directories that are located in your current directory.

  2. To change directories (on either OS) use the cd command followed by the directory name or path. Notice that you (typically) see the current directory name in the shell window prompt.

    If we need to move back out of our directory we can enter the command cd .. which will move up one directory.

  3. Finally, to make a directory, enter the command mkdir followed by the directory name to create a directory.
