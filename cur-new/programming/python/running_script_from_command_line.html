Shell Commands

To navigate to a particular python file we may need to make use of the following shell commands. (Whenever we say "enter a command" this means type the command and then press enter to execute it).

  1. The command ls allows you to view a list of all of the files and directories that are located in your current directory.

  2. To change directories use the cd command followed by the directory name or path. Notice that you (typically) see the current directory name in the shell window prompt.

    If we need to move back out of our directory we can enter the command cd .. which will move up one directory.

  3. Finally, to make a directory, enter the command mkdir followed by the directory name to create a directory.
