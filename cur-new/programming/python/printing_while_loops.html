Printing

The print statement will show (or print) a value to the command line. This is more or less the equivalent of "say" in snap. Print statements allow a programmer to print values and variables at a chosen location in the code. 


>>> x = 5
>>> print(x)
5

While Loop

Below is a count_up function written using a while loop. Side by side with the same code in snap. (We haven't learned to make functions yet in python, but this is an example.) 


i=0
while i < 5:
  print(i)
  i += 1

The while loop is analogous to the 'repeat until' block in snap However, in Python the loop ends when the condition becomes False. This is opposite of 'repeat until', which waits for its condition to become True.

Notice that the line with the while ends with a : and that the following lines are indented. The indentation indicates what code is repeated by the loop. The next line that is not indented will happen when the loop is over.
