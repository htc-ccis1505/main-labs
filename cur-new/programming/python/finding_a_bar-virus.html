Working with a Script

Get the starter file

Now it's time to seriously write some code. Start by right clicking on this link and select Save As. If you are working in the lab, save this to the C:\Python27 directory.

Head back to the shell and enter the dir command to make sure that the file made it into our Python27 directory. 


C:\Python27> dir
virus.py

Edit the File

We will be editing the python file in a text editor. There are many text editors available, any will do. In this lab we'll be using Brackets which should be installed the lab computers, and it is free to install on your home computers as well.

First, open up Brackets and then open virus.py. Make sure to read the comments at the top of the file and then take a brief glance at the code, but don't worry about it too much for right now. Also, make sure to set the variable called your_name just below the comments. For example, your_name = "Alonzo" (make sure your name is inside quotes as this is how Python designates text).

Head back to the shell and you can try running this file by entering python virus.py as shown below: 


C:\Python27> python virus.py

Even Numbers

Next try running the first exercise by entering python virus.py first_even_nums 5 (if it's working this should print the first 5 even numbers, starting with the number 2): 


C:\Python27> python virus.py first_even_nums 5
2
4
6
8

Uh oh...it looks like we've only printed the first 4 even numbers. Head back to the editor and look for Exercise 1 in the file. It is the function called first_even_nums. Update the code so that we get the correct behavior:
