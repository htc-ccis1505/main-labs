Let's try some example scripts using if and if-else.

  • Make a block that when given a light color, makes the sprite say the appropriate action. For example should say "go!"
  • Make a block that takes in a percentage and makes the sprite say the associated letter grade. For example, should say "C."

Pick one (or both) of the above blocks. If you used if statements to write the code, try to change it to use if-else blocks. If you used if-else statements, try to change it to use only if blocks.
