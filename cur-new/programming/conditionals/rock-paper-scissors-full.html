For this activity, you will create the Rock, Paper, Scissors game described on the previous page.

You may use any two costumes/sprites you like - it doesn't have to be the cat & dog. There are a lot of sprite "Costumes" that you can import from the Snap! File menu. The cat shown is "Cat 4" and the dog is "Dog 2b".

This script will use two different sprites. When the space bar key is pressed , each sprite will choose a random number to indicate rock, paper or scissors, and then say their choice. (It may be easiest to get the script working for the first sprite, and then duplicate that code for the second sprite.)

For a variable to be visible across many sprites, it is helpful to use a global variable instead of a script variable. However, you should prefer script variables when this behavior is not needed.

Once you have this working, add a third sprite. After the first two sprites pick their choice and say it, the third sprite should say who wins. To allow the sprites time to make their choice, have this third sprite wait for 1 second before deciding who wins.

You may find this chart useful.

Add two additional variables that keep track of how many times each character has won. You can make these variables visible on the stage by clicking on the check-box next to the variable name. Make sure that these reset to zero each time the green flag is clicked.

Requirements

The minimum requirements for this script are that it:
- starts when the green flag button is clicked
- sets a hidden variable for each sprite to a random number between 1 and 3 when the script starts and then uses that variable to determine which word each sprite should say
- sets visble variables to show how many times each sprite has won
