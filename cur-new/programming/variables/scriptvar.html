You can use variable to hold values that may change in your script, or you can use a variable for something that may stay the same, but is used in many places. Variables whose values don't change are called constants.

We often create constants to hold messages or other values, perhaps a tax rate or a phone number. By using a variable for that value, if it later needs to change, you only need to update the code in one place.
