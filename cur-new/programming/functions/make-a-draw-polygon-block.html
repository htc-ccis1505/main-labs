Now, you are going to make a block that takes two inputs.

Create a draw polygon block that takes as inputs:
- a number of sides
- the length of each side

In the block, call these inputs sides and length.

You should remember from this lesson how to draw a regular polygon. How many degrees did we turn altogether? How many times do we turn? What was the generalized pattern identified from the examples?

Testing is an important part of computer programming. Make sure to test out your new block with different values for length and number of sides. Testing allows you to catch your own errors before someone else finds them.
