Bug Dance!



Every good programmer goes through this at some point, and I think that you are ready.

You need to make a bug dance.

Requirements

This is a pretty open ended assignment, so have fun with it. However, don't feel like you need to choreograph a ballet or a music video.

The minimum requirements for this script are that it:
- starts when the green flag button is clicked
- uses at least 3 new custom blocks that you created
- each custom block must control an interesting sequence sprite movements
- the custom blocks should use input parameters to alter the movement sequence in some way
- sound is optional but encouraged

Load the starter project

Load this snap project. (Or, this is the same thing without the big sound file).

We've made some starter custom blocks for you, in the "Looks" palette. Definitely edit... them to see what we've done. We've also included a few examples of how to put them together for a dance. Try it out.

Improve the Script

Make at least 3 custom blocks to do complicated steps, or to repeat sets of steps in ways that you'll do over and over. Add inputs to your custom blocks to change your steps in interesting ways.

We've included several costumes in the project that you should use in new steps. Look at how we used the costumes in the custom blocks we already created. You could also import your own sounds if you would like. Go to the Sounds tab above the scripting area and drag a sound from your computer.

Have fun with this!

Share!

This is another program that you are welcome to share in the chatroom. For this assignment, sharing your dancing bug is officially OK. However, it's never OK to copy or submit someone else's work as your own. I expect everyone's bug dance to be different and unique.

Export and Submit

When everything works as you expect, export your script (refer back to the first inclass lab if you need help) and upload that to the lab dropbox in D2L.
