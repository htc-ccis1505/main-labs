Field of Flowers

Time to go a little bit crazy...

We are going to use the blocks we created earlier to write a script that will draw a field of random flowers.

Requirements

This is a pretty open ended assignment, so have fun with it.

The minimum requirements for this script are that it:
- starts when the green flag button is clicked
- adds at least two new custom blocks that use the simpler blocks created earlier
- uses repeat to draw multiple flowers filling the screen
- incorporates randomness to draw a different field of flowers each time the script is run

Starter File

This starter project, contains a few fun backgrounds. To use them, click on the Stage sprite at the lower right of the snap window, select the Costumes tab at the top of the scripting area, and select the background that you want for your program.

Composition with Custom Blocks

Using one custom block inside another custom block is called composition. We are going to use the simple blocks we made earlier to make more complex blocks to draw different types of flowers.

Use some of the simple custom blocks you've already written, to create at least two new custom blocks that make use of those simpler blocks to do complicated things.
