Draw Random Shapes

Let's have a little bit of fun...

Use the blocks created earlier to write a script that will draw a random shapes of various sizes all over.

Requirements

This is a pretty open ended assignment, so have fun with it.

The minimum requirements for this script are that it:
- starts when the green flag button is clicked
- uses the draw polygon block to draw random shapes of random sizes
- uses a repeat to draw multiple shapes
- uses randomness to draw a shapes in different positions filling the screen

Once you have that minimum requirements working, try to enhance the program by drawing in different colors. You might also experiment with rotating and/or combining the shapes. Remember those fancy screen-saver programs? Be creative, the sky is the limit.
