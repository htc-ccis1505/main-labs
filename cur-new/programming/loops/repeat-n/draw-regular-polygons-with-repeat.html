Remember that a regular polygon is a shape in which all the sides are the same length and all the turning angles are the same. This makes them easy to draw using a repeat.

Use what you learned from drawing a square and use the to draw the following regular shapes:
- Equilateral Triangle (3 sides)
- Pentagon (5 sides)
- Hexagon (6 sides)
- Octagon (8 sides)

Save the scripts as you go because, in a following activity, you'll be trying to figure out a pattern.

As you draw each shape, think about the relationship between the number of sides and the angle that you turn. As you have more sides, does the angle increase or decrease?

Challenge
As you draw polygons with more and more sides, you are getting closer and closer to a circle. Is there a point at which it is actually (mathematically) a circle? Can you modify the script to make a circle?
