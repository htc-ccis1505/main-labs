Look at the following script to draw a square. Notice that there is a move and turn block to draw each side of the square. Those blocks are repeated four times - once for each side.

We can use the block to make drawing shapes like this square a lot easier!

Not only is there less code, it's also easier to read and understand. You can immediately see that all four sides are the same. This also helps to eliminate bugs. You can't get one side longer than the other by accidentally putting in a different number in one of the move blocks, or create a different shape by using a different number in one of the turn blocks.

There are three different numeric values in the script—4, 100, and 90—that you can think of as parameters that change the resulting picture. Play with the script to understand how each number contributes to the "squareness" of the shape that is drawn. It is important to understand this now, as you'll be using this pattern to make different shapes throughout the lab.
