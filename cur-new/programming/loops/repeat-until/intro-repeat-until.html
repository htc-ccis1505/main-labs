In that code we used a forever and used an if statement with the stop script block to stop the loop. This is functional code, but it can be a little hard to identify exactly how the program ends. It's easy to see the forever loop and assume that the program will never stop.

You should try to avoid writing code that uses the forever block, and instead use one of the other types of loops where the ending condition is more clear.

Be careful with the ask and answer blocks. The ask will always hold the answer to the most recently asked question. While that is relatively clear here, because there is so little code, it does not take much for it to become confusing. It is recommended that you always save the answer into a clearly named variable immediately following the ask block. That way you will never wonder what the answer contains.
