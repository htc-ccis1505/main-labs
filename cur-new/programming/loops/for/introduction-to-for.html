Repeating a part of your program is a very common practice in programming. Sometimes, though, you want to do almost the same thing for a certain number of times, but with a slight variation.

For instance:
- You want to draw 20 squares on the screen, but each a different size.
- You want to draw 100 polygons with different numbers of sides.

These situations can often be handled by the for block, which is found near the bottom of the Control palette:

NOTE: in the current version of snap, the for block isn't installed automatically. You will need to import it once for each new snap project in which you use it. To do so, click on the icon at the top of the screen and select Import Tools.)
