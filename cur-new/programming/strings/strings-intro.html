In programming, words and sentences are often called strings. A string is just a sequence of characters. Those characters might be anything - letters, spaces, numbers, punctuation, etc.

In this lab we will practice working with variables, loops, and conditions while learning to work with strings.

Snap has many blocks that allow us to work with strings. You have seen some of these already, such as the block which is used to join two values together. We can use this block to join to words or sentences (strings) with variables and the values that come from reporter blocks.

When you use the join block to put values together, you need to add spaces at the beginning and end of words or sentences to make them look nice.

By using the join block, we can put together any number of values by clicking on the arrows on the right side of the block to add additional input slots.

Do you see the little dot after the word "Hello" in the example above? That is a space character. Since you cannot typically see a space character, it is shown in the input as a little dot so that you know it is there.

It's important to use spaces in the join block when joining variables into sentences so that the messages and the variables are not squished together. We want to see this code say "Hello Alonzo" with the space between words, not "HelloAlonzo" without a space.
