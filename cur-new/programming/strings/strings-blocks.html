The Snap blocks for working with Strings are found on the Operators palette. You can probably guess what they will do, just by looking at them.

Get a character

The block will give you back a single character from the string. In this case, the block would report w since that is letter 1 of world.

Length

The block will tell you how many characters are in the string. In this case, the block would report 5 since that is the length of world.

Type

The block will tell you if a value is a number. However you can also use it to identify other types as well by using the dropdown menu next to number. One of the other options is text. So this block can also be used to tell you if something is text instead of a number.

Experiment with the is () a () ? block to see what type of values it considers text and what it does not.
